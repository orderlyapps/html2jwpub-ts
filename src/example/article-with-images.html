Article with Multimedia Content

This example demonstrates how to include images in your HTML content. The converter will process these images and include them in the .jwpub publication.

Including Images

Images should be stored in a folder named article-with-images_files or article-with-images.html_files in the same directory as this HTML file.

Multiple Images

You can include multiple images throughout your document. Each image will be processed and included in the publication database.

Image References

The converter automatically detects media files referenced in your HTML and includes them in the .jwpub package. Supported formats include:

  • JPEG/JPG images
  • PNG images
  • GIF images
  • SVG graphics

Make sure all referenced media files exist in the appropriate _files folder before conversion.

Best Practices

  • Use descriptive file names for images
  • Always include alt text for accessibility
  • Optimize images for web to reduce file size
  • Use appropriate image formats (JPEG for photos, PNG for graphics)
