1. Introduction

Welcome to this comprehensive guide on converting HTML files to the JWPUB publication format. This multi-section article demonstrates how larger documents with multiple sections can be effectively converted.

The JWPUB format is used for digital publications and includes:

  • Encrypted content storage
  • Embedded multimedia support
  • Structured database format
  • Compressed packaging

2. Getting Started

2.1 Prerequisites

Before you begin, ensure you have the following:

  1. HTML files prepared for conversion
  2. Any associated media files organized in _files folders
  3. Publication metadata (title, symbol, year)

2.2 File Organization

Organize your files following this structure:

my-publication/
├── article1.html
├── article1_files/
│   ├── image1.jpg
│   └── image2.png
├── article2.html
└── article2_files/
    └── diagram.svg
Note: The converter automatically detects media files in folders named [filename]_files or [filename].html_files.

3. Features

3.1 Content Encryption

All HTML content is encrypted using AES-128-CBC encryption with zlib compression. The encryption key is derived from your publication metadata.

3.2 Database Structure

The converter creates a SQLite database with the following key tables:

  • Publication - Publication metadata
  • Document - Encrypted HTML content
  • Multimedia - Media file references
  • PublicationViewItem - Navigation structure

3.3 Manifest Generation

A manifest.json file is automatically generated with:

  • SHA-256 content hash
  • Publication metadata
  • Timestamp information
  • File size and compression details

4. Best Practices

4.1 HTML Guidelines

  1. Use semantic HTML5 elements
  2. Include proper headings hierarchy (h1, h2, h3)
  3. Add alt text to all images
  4. Validate HTML before conversion

4.2 Media Guidelines

  • Optimize images before adding them
  • Use appropriate formats (JPEG for photos, PNG for graphics)
  • Keep file sizes reasonable
  • Use descriptive file names

4.3 Publication Metadata

Field Description Example
Title Publication title Watchtower 2022
Symbol Publication code w22
Year Publication year 2022
Language Index MEPS language code 1 (English)

5. Troubleshooting

5.1 Common Issues

Media Files Not Found

Problem: Images don't appear in the converted publication.

Solution: Ensure media files are in a folder named [filename]_files and paths in HTML match exactly.

Conversion Fails

Problem: The conversion process stops with an error.

Solution: Check that:

  • HTML files are valid
  • All required metadata is provided
  • File paths don't contain special characters

Large File Size

Problem: The generated .jwpub file is too large.

Solution: Optimize images and remove unnecessary content or inline styles.

Tip: Always test with a small sample file first before converting large publications.
